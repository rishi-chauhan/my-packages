# Contributing to image_array_and_histogram

First off, thank you for considering contributing to this project! Any and all contributions are welcome.

## How Can I Contribute?

### Reporting Bugs

If you find a bug, please open an issue on our [issue tracker](https://github.com/rishi-chauhan/my-packages/issues).

Please include the following information in your bug report:

*   A clear and descriptive title.
*   A description of the bug, including the steps to reproduce it.
*   The expected behavior and what actually happened.
*   Your Python version and the version of this package.

### Suggesting Enhancements

If you have an idea for a new feature or an improvement to an existing one, please open an issue on our [issue tracker](https://github.com/rishi-chauhan/my-packages/issues).

Please include the following information in your enhancement suggestion:

*   A clear and descriptive title.
*   A detailed description of the proposed enhancement.
*   Any relevant code snippets or examples.

### Pull Requests

If you'd like to contribute code, you can open a pull request.

1.  Fork the repository.
2.  Create a new branch for your feature or bug fix.
3.  Make your changes and commit them with a clear and descriptive commit message.
4.  Push your changes to your fork.
5.  Open a pull request to the `main` branch of this repository.

Please make sure your code follows the existing code style and that you've added or updated tests as appropriate.

## Styleguides

This project follows the [PEP 8](https://www.python.org/dev/peps/pep-0008/) style guide for Python code.
